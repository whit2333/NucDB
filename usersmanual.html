User's Manual

Introduction

NucDB is an attempt to easily interface and search experimental data, add data to the database, and make pretty plots. It is based on the ROOT libraries.

Experiments

This project begain as an attempt to consolidate the world data on spin structure functions and make nice plots of them. Naturally the unpolarized structure functions were added.
